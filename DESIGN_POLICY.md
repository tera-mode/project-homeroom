## 1. ターゲットと絵柄の方向性

### ターゲット
**日本の小学校高学年〜中学生（10〜14歳）** をメインターゲットとする。
サブターゲットはインディーゲーム好きな大人。

### 絵柄の一言コンセプト
> **「コロコロ × マンガっぽい × ちょっとかわいい」**

具体的に意識するスタイル：
- **コロコロコミック / ちゃお** のような、太くはっきりとした輪郭線
- **どうぶつの森 / スーパーマリオ** のような、誰でも親しみやすいキャラデザイン
- **ポケモンのゲーム内スプライト**（ドット絵の読みやすさ・個性の強さ）
- **よりもい / クラスのきみ** 系の、明るくポップな学園カラーリング

### 避けるべきスタイル
- ❌ リアル系・写実的なグラフィック
- ❌ ダーク・ホラー的な色使い
- ❌ 美少女ゲーム的なセクシー表現
- ❌ ちびキャラでないロングプロポーションのキャラ
- ❌ 過度に「大人っぽい」洗練されたデザイン（小学生に刺さらない）

## 2. キャラクタービジュアル仕様

### スプライトサイズ
- **ゲーム内表示**：48×48 px（グリッドのマス内に収まるサイズ）
- **画像生成時**：192×192 px → ゲーム内でダウンスケール
- **カード表示用**：128×128 px（転校生紹介カード用）

### キャラクタースタイル

#### 基本ルール
- **ちびキャラ（2頭身〜2.5頭身）** を原則とする
- 頭が大きく、体が小さい「かわいい」プロポーション
- **太い輪郭線**（線の太さはキャラサイズの約4〜6%を目安）
- **フラットカラー** 主体。複雑なグラデーションは使わない
- **影は1段階のみ**（または影なし）。アニメ塗り
- 表情は**大きく誇張**する（喜怒哀楽がひと目でわかること）

#### 制服・見た目
- 全キャラ共通：夕凪学園の制服ベース（白シャツ + 紺のスラックス or スカート）
- キャラの個性は**髪型・髪色・小物・制服のアレンジ**で表現する
  - 例：「ケンカ大将」は制服を崩している、「学園のマドンナ」はリボンが豪華など

#### 配色パレット（キャラクター用）
小中学生が好む「明るくポップな色味」を優先する。

| 用途 | 推奨色 | 補足 |
|---|---|---|
| 肌色（明るめ） | `#FFD5B5` | 少し黄味がかったアニメ標準肌色 |
| 肌色（やや濃い） | `#F5C299` | 多様性を意識した第2肌色オプション |
| 輪郭線 | `#2D1A0E` | 純黒より少し茶寄りで柔らかく |
| 制服・紺 | `#2E4080` | 深すぎない明るめネイビー |
| 制服・白 | `#F8F8FF` | 純白でなく少し温かみのある白 |
| 髪色（一般） | 黒`#2B1B14`・茶`#8B5E3C`・金`#F2C55E` | |
| アクセントカラー | 各キャラ個性に合わせて設定 | |

## 3. UI / 画面デザイン仕様

### 全体テーマ
**「放課後のノート」** スタイル。

- 背景は**方眼紙または罫線ノート**のテクスチャ
- UIパーツは**手書き風の枠線**・**スタンプ風**のアイコン
- フォントは**丸ゴシック**系（角ばったゴシックは避ける）
- ポップアップは**ふきだし風**のデザイン

### カラーパレット（UI全体）

| 用途 | カラー | HEX |
|---|---|---|
| 背景（ベース） | ノート白 | `#FAFBF0` |
| 背景（サブ） | クリーム | `#F5F0DC` |
| 方眼線 | ライトグレー | `#D0D8C0` |
| メインアクセント | スカイブルー | `#5BAADC` |
| サブアクセント | コーラルピンク | `#FF7F7F` |
| テキスト（メイン） | ダークブラウン | `#3A2A1A` |
| テキスト（サブ） | グレイッシュブラウン | `#7A6A5A` |
| 成功・プラス | グリーン | `#5BAE5F` |
| 失敗・マイナス | レッド | `#E05050` |
| ボタン（プライマリ） | ブルー | `#4A90D9` |
| ボタン（セカンダリ） | イエロー | `#F5C842` |
| シャドウ | セミトランスペアレント | `rgba(0,0,0,0.15)` |

### パラメータバー

- 5種のパラメータバーを画面端（右側 or 上部）に常時表示
- 各バーは対応するアイコン + 名前 + 数値 + 塗りバー
- 数値が増えるときは **ポップアップアニメーション** `+2!` （黄色・ポップ体）
- 数値が減るときは **ぐわっとアニメーション** `-1!` （赤・揺れエフェクト）

### グリッド（4×4教室）

- 各マス：**64×64 px** 以上（クリック・タップしやすいサイズ）
- 空席：明るいクリーム色の背景 + 点線の枠
- 占有席：キャラクタースプライト表示
- シナジー発動中：マスの周囲が**きらきらエフェクト**（黄色・星形）
- クラッシュ発動中：マスの周囲が**もやもやエフェクト**（紫・雲形）
- 配置プレビュー中：対象マスが**青いハイライト** + パラメータ差分ポップアップ

### 転校生カード

カードデザインは「ガチャカード」をイメージ。

```
┌──────────────────────┐
│ ★★ アイドルタイプ    │  ← レア度 + カテゴリ
│                      │
│      [スプライト]     │  ← 128×128px
│                      │
│   星宮リナ           │  ← 名前（大きめフォント）
│   恋愛+2 / 人気+1    │  ← 主要効果（色付きアイコン）
│   ━━━━━━━━━━━━━━━━━━│
│  「前の学校でも注目…」│  ← ヒントテキスト（小文字）
└──────────────────────┘
```

- レア度はカードの枠色で表現：★→シルバー、★★→ゴールド、★★★→虹（グラデ）
- 出現時は**スライドインアニメーション** + **キラキラエフェクト**

## 4. サウンドデザイン方針

### BGM
- 主調：**学園ポップ / 明るいピアノ + 電子音**
- 時間帯で変化：朝（明るく元気）→ 放課後（やや落ち着いた）→ 夜（少ししっとり）
- ループ素材はFree Music Archive / MusMus から調達（商用利用可のもの）

### SE（効果音）
| 場面 | 音のイメージ |
|---|---|
| 転校生出現 | キラーン！（明るい高音） |
| 席に配置 | ポン！（軽い打音） |
| シナジー発動 | きらきらきら〜（上昇音） |
| クラッシュ発動 | ドン…（少し低めの音） |
| パラメータ上昇 | チャラン！（達成感のある音） |
| パラメータ下降 | ガクン（短い下降音） |
| 隠し特性発動 | ドドドン！ → キラキラ（驚き → 発見） |

## 5. フォント指定

| 用途 | フォント | 備考 |
|---|---|---|
| タイトル | **Rounded M+ 1c** または **Kosugi Maru** | 丸ゴシック・日本語対応 |
| 本文・UI | **Noto Sans JP** | 可読性重視 |
| 数値・パラメータ | **M+ 1m** または **JetBrains Mono JP** | 等幅で数字が読みやすい |
| ポップアップ（+2!など） | タイトルと同じ丸ゴシック | 太字・大きめ |

すべてGoogleフontsまたはOFLライセンスで商用利用可能なものを使用すること。

## 6. アニメーション仕様

| アニメーション | 内容 | 時間 |
|---|---|---|
| 転校生カード出現 | 下からスライドイン | 0.3秒 |
| 配置決定 | キャラがマスにパコン | 0.2秒 |
| シナジー発動 | 星のパーティクルが広がる | 0.5秒 |
| パラメータ変動 | 数値がポップして元に戻る | 0.4秒 |
| 日替わり | フェードアウト → フェードイン | 0.5秒 |
| 隠し特性発動 | カードが光る → 新情報が現れる | 1.0秒 |
| エンディング | スライドショー風 + テキスト | 演出重視 |

アニメーションは **Godot の AnimationPlayer または Tween** で実装する。
アニメーション時間は「テンポよく、もたつかない」を最優先にする。

---

## 7. アクセシビリティ

- フォントサイズ：**最小16px** をUIテキストの下限とする
- カラーコントラスト：テキストとbg間は**4.5:1以上**（WCAG AA準拠）
- タップ/クリックターゲット：**最小44×44px**
- シナジー/クラッシュの区別は**色だけでなくエフェクトの形**でも識別できるようにする（色覚多様性対応）
